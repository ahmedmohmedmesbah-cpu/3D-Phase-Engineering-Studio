3D Phase Engineering Studio — 3D Viewer Help

This page explains how to publish and use your free 3D viewer on GitHub Pages.

  1. Create a public repository named YOUR-USERNAME.github.io on GitHub.
  2. Upload all files from the ZIP (keep the folder structure).
  3. Go to Settings → Pages and set Source to Deploy from a branch, Branch main, folder /(root).
  4. Visit https://YOUR-USERNAME.github.io.

Show your model

  • Put your file in models/ and rename it to project.glb.
  • Or click Open local .glb/.gltf to load a file from your computer (no upload).

Controls

  • Rotate: left mouse
  • Pan: right mouse
  • Zoom: wheel
  • Reset: the button in the top bar
